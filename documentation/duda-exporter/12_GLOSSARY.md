# Glossary

## Domain Terms
| Term | Definition |
|------|------------|
| **Duda** | The website builder platform we are targeting. |
| **Sitemap** | `sitemap.xml` file listing all valid pages on a site. |
| **Worker** | The background Node.js process that runs Playwright. |
| **Job** | A single request to export a website. |

## Technical Terms
| Term | Definition |
|------|------------|
| **Playwright** | Browser automation library. |
| **BullMQ** | Job queue library based on Redis. |
| **Headless** | Running a browser without a visible UI. |
