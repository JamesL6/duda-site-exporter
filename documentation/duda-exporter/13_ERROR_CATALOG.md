# Error Catalog

## Error Codes
| Code | Message | Resolution |
|------|---------|------------|
| `JOB_FAILED_TIMEOUT` | Scrape took too long. | Reduce concurrency or retry. |
| `INVALID_URL` | URL is not a valid Duda site. | Check URL and try again. |
| `SITEMAP_NOT_FOUND` | Could not find sitemap.xml. | Verify site is public. |
| `AUTH_ERROR` | Not logged in. | Redirect to login. |
